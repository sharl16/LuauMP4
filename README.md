# LuauMP4 - A <u>free<u> solution to playing videos on ROBLOX

Last Updated: 6/1/2026 (v2026.1)

A ROBLOX Plugin that allows you to playback videos for free by splitting up JPG's or PNG's in many ImageLabels and playing them sequentially.
- Get the Plugin at: https://create.roblox.com/store/asset/18735535757, or use it locally by extracting this repository.
